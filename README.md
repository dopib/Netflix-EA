## 1. Цели анализа

* Провести предварительную очистку и валидацию данных.

* Подготовить данные для дальнейших задач (классификация, рекомендация, временные ряды).

## 2. Описание данных

Файл ноутбука ожидает CSV/TSV/Excel с информацией о заголовках Netflix. В подобных датасетах обычно встречаются следующие колонки:

* `show_id` — уникальный идентификатор
* `type` — тип (`Movie` / `TV Show`)
* `title` — название
* `director` — режиссёр
* `cast` — актёрский состав
* `country` — страна производства
* `date_added` — дата добавления на платформу
* `release_year` — год выпуска
* `rating` — возрастной рейтинг
* `duration` — продолжительность (минуты или сезоны)
* `listed_in` — жанры / категории
* `description` — краткое описание

Если в вашем конкретном файле колонки отличаются, раздел "Чтение и первичный осмотр" ноутбука содержит код для быстрой проверки и переименования столбцов.

## 3. Установка и запуск

Рекомендуется использовать виртуальное окружение.

```bash
# создать окружение (опционально)
python -m venv .venv
source .venv/bin/activate  # Linux / macOS
.\.venv\Scripts\activate   # Windows

# установить зависимости
pip install -r requirements.txt

# или установить минимальный набор вручную
pip install jupyter pandas numpy matplotlib seaborn plotly

# запустить ноутбук
jupyter notebook netflix-shows-and-movies-exploratory-analysis.ipynb
```

Если предпочитаете Google Colab — загрузите ноутбук в Colab и запустите клетки.

## 4. Краткое описание разделов ноутбука

* **Импорты и настройки** — загрузка библиотек и стиль визуализаций.
* **Чтение данных** — загрузка файла, быстрый просмотр, проверка типов и пропусков.
* **Очистка** — обработка пропусков, нормализация столбцов (даты, продолжительности), разбор нескольких жанров в `listed_in`.
* **EDA (Exploratory Data Analysis)** — распределения по годам, типам контента, топ-страны/режиссёры/актёры, частотный анализ жанров.
* **Визуализации** — гистограммы, графики времени, тепловые карты, barplots для топ-10 и т.п.
* **Выводы** — краткие инсайты и наблюдения.
* **Экспорт подготовленных данных** — сохранение очищенных таблиц и/или артефактов визуализаций.

## 5. Ключевые находки (примеры того, что стоит искать)

* Возрастные тренды: увеличение/уменьшение количества релизов по годам.
* Какие страны доминируют по количеству контента.
* Отношение `Movie` vs `TV Show` по годам добавления на платформу.
* Наиболее частые жанры и их сочетания.
* Как изменялось распределение рейтингов (PG-13, TV-MA и т.д.).
